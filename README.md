# ESP32 ULP 1-Wire example
Using GIPO 32 (RTC_GOIO 9) for 1wire. A pull-up 4kohm on 3.3v between GPIO 32 and a magnetic Temperature sensor DS18B20 waterproof 1m Cable at the end.

## Example output

Note: GPIO15 is connected to GND to disable ROM bootloader output.
